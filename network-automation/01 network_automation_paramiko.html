3.1 模块介绍

Paramiko 是什么？

📌 核心定位

  • ▸ 纯 Python 实现，无需额外依赖 OpenSSH 客户端
  • ▸ 支持口令认证和公钥认证两种方式
  • ▸ 提供 SSH 客户端和服务器功能
  • ▸ 实现安全的远程命令执行和文件传输

💡 名称由来

由 Robey Pointer 开发，名称来自 "paramiko"（具有怪异性参数的 SSH），是 Python 实现 SSHv2 协议的第三方模块。

🔍 与 Telnetlib 对比

Telnetlib 不推荐
Paramiko 强烈推荐
安全性
明文 ⚠️ 加密 ✅
认证方式
仅密码 密码+公钥
功能扩展
仅命令 命令+文件+端口转发
生产环境
不安全 企业级

安装 Paramiko

推荐使用国内镜像源加速安装

步骤 1

基础安装

# 默认安装（可能较慢）
pip install paramiko

# 国内镜像加速（推荐）
pip install paramiko -i \
  https://mirrors.aliyun.com/pypi/simple/ \
  --trusted-host mirrors.aliyun.com
步骤 2

验证安装

import paramiko
print(paramiko.__version__)
# 输出: 2.7.2 或 3.x.x

✅ 安装成功标志：无报错且能输出版本号

⚠️ 常见问题

❌ 报错：Microsoft Visual C++ 14.0 is required

原因：缺少编译环境

解决：pip install --only-binary :all: paramiko

❌ 报错：No module named 'cryptography'

原因：依赖库未自动安装

解决：pip install cryptography paramiko

❌ 连接超时或拒绝

原因：设备未开启SSH或防火墙拦截

解决：检查设备SSH服务与网络连通性

安全策略

主机密钥策略

set_missing_host_key_policy() · 应对未知主机

✅

AutoAddPolicy

推荐

自动添加主机名及主机密钥到本地 HostKeys 对象，不提示确认

client.set_missing_host_key_policy(
  paramiko.AutoAddPolicy()
)

🎯 适用场景：自动化脚本、批量运维

⚠️

WarningPolicy

调试

记录未知主机的 Python 警告，允许连接但提示新连接

client.set_missing_host_key_policy(
  paramiko.WarningPolicy()
)

🎯 适用场景：开发调试阶段

🚫

RejectPolicy

默认

自动拒绝未知主机名和密钥，依赖 load_system_host_keys

client.set_missing_host_key_policy(
  paramiko.RejectPolicy()
)

🎯 适用场景：高安全环境、手动管理

📚 教材第140页：三种策略的详细对比与代码示例

实战代码

密码认证连接示例

最基础的 Paramiko 连接方式

import paramiko
import time

# 1. 创建客户端实例
client = paramiko.SSHClient()

# 2. 设置自动添加未知主机密钥
client.set_missing_host_key_policy(
  paramiko.AutoAddPolicy()
)

# 3. 建立连接
client.connect(
  hostname='192.168.56.100',
  port=22,
  username='admin',
  password='Huawei@123',
  allow_agent=False,
  look_for_keys=False
)

# 4. 执行命令
stdin, stdout, stderr = client.exec_command(
  'display version'
)
output = stdout.read().decode('utf-8')
print(output)

# 5. 关闭连接（必须！）
client.close()

🔍 代码解析

SSHClient()

创建会话对象

AutoAddPolicy()

免交互确认新主机

allow_agent=False

禁用ssh-agent干扰

look_for_keys=False

不搜索本地私钥

⚠️ 异常处理

• AuthenticationException → 用户名/密码错误

• SSHException → SSH协议错误

• socket.timeout → 网络不可达

高级用法

交互式 Shell

invoke_shell() · 处理分页与长输出

🎯 核心问题

--More-- 分页陷阱

华为设备默认分页显示，exec_command() 只能获取第一页内容！

✅ 解决方案

# 关键：设置不分页
channel.send("screen-length 0 temporary ")

# 启动交互式shell
channel = client.invoke_shell()

# 发送命令
channel.send("display current-configuration ")

# 循环读取输出
output = ""
while channel.recv_ready():
  output += channel.recv(65535).decode()

📊 exec_command vs invoke_shell

特性exec_commandinvoke_shell
适用场景短命令长输出/交互
分页处理❌ 困难✅ 容易
返回类型三元组Channel对象
复杂度简单较复杂

💡 最佳实践

  • • 巡检用 exec_command + screen-length 0
  • • 配置备份用 invoke_shell
  • • 必须 time.sleep() 等待设备响应
  • • 生产环境封装异常处理与重试机制

NetworkInspector 类设计 · OOP封装

class NetworkInspector:
  """网络设备巡检器"""

  def __init__(self, device_info):
    self.device = device_info
    self.client = None
    self.results = {}

  def connect(self):
    self.client = paramiko.SSHClient()
    self.client.set_missing_host_key_policy(
      paramiko.AutoAddPolicy())
    try:
      self.client.connect(...)
      return True
    except Exception as e:
      print(f"[失败] {e}")
      return False

  def run_command(self, cmd):
    stdin, stdout, stderr = \
      self.client.exec_command(cmd)
    return stdout.read().decode()

  def inspect(self):
    items = {
      "version": "display version",
      "cpu": "display cpu-usage",
      "memory": "display memory-usage",
      "device": "display device",
      "clock": "display clock",
      "log": "display logbuffer"
    }
    for name, cmd in items.items():
      self.results[name] = self.run_command(cmd)

扩展功能

批量巡检多台设备

循环遍历 · 异常隔离 · 汇总报告

def batch_inspect(devices_list):
  all_results = []

  for device in devices_list:
    inspector = NetworkInspector(device)

    try:
      results = inspector.inspect()
      if results:
        inspector.save_report()
        all_results.append({
          "device": device["name"],
          "status": "success"
        })
    except Exception as e:
      all_results.append({
        "device": device["name"],
        "status": "failed",
        "error": str(e)
      })
    finally:
      inspector.disconnect()
      time.sleep(2) # 间隔

  # 汇总输出
  success = sum(1 for r in all_results
            if r["status"] == "success")
  print(f"成功: {success}/{len(all_results)}")
